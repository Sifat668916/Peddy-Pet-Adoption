# Peddy - Pet Adoption Platform

Peddy is a modern, responsive pet adoption platform designed to connect loving families with their perfect pet companions. Built with a sleek **Bento Grid** theme, the application provides a modular and intuitive user experience for browsing available pets, filtering by categories, and managing a "Liked Pets" list.

## 🐾 Features

- **Bento Grid Design**: A clean, modular layout that organizes content into structural "boxes" with soft rounded corners and deep shadows.
- **Dynamic Category Filtering**: Browse pets by type (Cats, Dogs, Rabbits, etc.) with real-time API integration.
- **2-Second Loading State**: A mandatory polished loading spinner to ensure content transition remains smooth.
- **Sorting by Price**: Quickly find pets within your budget using the descending price sort feature.
- **Liked Pets Sidebar**: A dedicated sidebar to keep track of your favorite pets while browsing.
- **Adoption Countdown**: A formal 3-second countdown modal that marks a pet as "Adopted" upon completion.
- **Detailed Descriptions**: Pop-up modals providing vaccination status, breed info, gender, and full personality details for each pet.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Technologies Used

- **HTML5**: Semi-semantic structure.
- **Tailwind CSS 4**: Functional modular styling and layouts.
- **Vanilla JavaScript**: Pure logic for DOM manipulation and API handling.
- **Vite**: Ultra-fast development server.
- **Font Awesome**: iconography.
- **Bento Aesthetic**: Custom CSS variables for the board-style design.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (LTS version recommended)

### Installation
1. Clone the repository:
   ```bash
   git clone <your-repository-url>
   ```
2. Navigate to the project folder:
   ```bash
   cd <project-folder>
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally
To start the development server, run:
```bash
npm run dev
```
Then open your browser to the URL shown in your terminal (usually `http://localhost:3000`).

## 📁 Project Structure

- `index.html`: The main entry point and structural layout.
- `src/script.js`: Core logic for API fetching, sorting, and UI interactions.
- `package.json`: Project metadata and dependencies.
- `vite.config.ts`: Configuration for the Vite build tool.
